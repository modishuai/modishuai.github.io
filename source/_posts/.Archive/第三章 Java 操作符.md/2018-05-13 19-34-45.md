# Java 操作符

## 算术操作符
`+、-、*、/、%`
##  自动递增和递减
`++、--`
前缀式: ++a，先执行运算，再生成值。
后缀式: a++，先生成值，再执行运算。
## 关系操作符
`<、>、<=、>=、!=`
## 逻辑操作符
`与( && )、或( || )、非( ! )`
**短路**
在使用逻辑操作符的过程做，会遇到 *短路* 现象，也就是一旦能够明确无误的确定表达式的值，就停止计算剩余表达式的值。
## 直接常量
`long i = 1L;`
直接常量的后缀字符标志着他所属类型。
## 按位操作符
按位操作符用来操作整数基本数据类型中的单个“bit”，二进制位。按位操作符会对两个参数中对应的位执行布尔代数运算，生成最终结果。
