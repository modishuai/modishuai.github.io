Java中的引用传递示例分析

发表于 2018-07-25 更新于 2019-12-10 分类于 后端笔记

对象的引用存放在栈内存，其对象存储在堆内存中。

引用传递的核心意义：同一堆内存空间可以被多个栈内存所指向（1堆 : N栈），不同的栈内存可以对同一堆内存进行内容修改。

简单 Java 类编写原则：

  • 类名称 = 表名称

  • 属性名称（类型） = 表字段 （类型）

  • 一个实例化对象 = 一行记录

  • 多个实例化对象（对象数组）= 多行记录（外键）。

  • 引用关系 = 外键约束
