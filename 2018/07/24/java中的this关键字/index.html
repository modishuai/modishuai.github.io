实现类方法调用

  • 普通方法

    普通方法调用前加或不加this没有必须要求，但是为了方法调用的严谨性，通常加上this关键字靠谱。

    在一个类中除了普通方法还有构造方法，多个构造方法间进行互相调用this(arguments)。

  • 构造方法

    • 构造方法互相调用一定要放在第一行。

    • 构造方法 互相调用一定要留出口（不要循环调用）。

表示当前类对象

总结

  • 类中的构造方法间的互相调用，不仅要放在第一行还要保留出口。

  • 类中的属性调用为避免作用域带来的问题，要使用 this 关键字。

  • this 表示当前对象，指的是当前正在调用类中方法的对象，this不是一个固定的。
